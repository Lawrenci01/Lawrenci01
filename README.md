## Projects

### 📱 Abaca Scanner
> Mobile app that scans abaca leaves using a phone camera and identifies **Retting Harvest Stage (RHS)** color using a trained ML/CV model.

- Real-time leaf classification via on-device model
- Helps farmers assess abaca fiber quality instantly
- **Stack:** Python · Mobile (Camera API) · ML/Computer Vision

---

### 🌦️ Abaca Climate Predictor
> Predicts **CO₂ levels, temperature, and humidity** in abaca plantations using deep learning and ensemble models.

- Models: **LSTM**, **Triple MLP**, **Ensemble**
- Daily forecasting to support crop monitoring decisions
- **Stack:** Python · LSTM · MLP · Ensemble Learning · Data Science
